Stack

The stack is a last in , first out data structure that performs operations to data on the top of the stack.

Queue

The queue is a first in, first out data strucutre that can be used to model a variety of situations.

Linked List

The linked list is a dynamic data structure that consists of nodes which link to one another.

Tree

The tree is a hierarchical data structure in which nodes have children and parents.

Binary Tree

The binary tree is a special case of the tree data structure where each node can have at most two child nodes.

Graph

A graph is a data structure that consists of a collection of edges and vertices.

Array

An array is a simple data structure that can store groups of objects of the same or different types.

Hash Table

A hash table is table that uses a hash function to make operations on the table very fast.
