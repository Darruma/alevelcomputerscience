Bubble Sort

Bubble Sort is an inplace sorting algorithm that sorts by bubbling items up the array.

Insertion Sort

Insertion Sort is an inplace simple sorting algorithm that works the way we sort playing cards in our hands.

Merge Sort

Merge Sort is a divide and conquer algorithm. It divides the array up and into pieces and merges them into order.

Quick Sort

Quick Sort is a divide and conquer algorithm that picks an element as a pivot and partitions the given array around the picked pivot.

Linear Search

Linear Search is a simple searching algorithm. It sequentially checks each element of the list for the target value.

Binary Search

Binary Search is very efficient searching algorithm that requires the array to be sorted.

Dijkstra's Algorithm

Dijkstra's Algorithm is an algorithm that computes the shortest paths between nodes in a graph.

A* Pathfinding

A* is a pathfinding algorithm that uses a heuristic in order to find the shortest path between nodes.
